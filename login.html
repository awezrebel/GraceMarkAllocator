Grace Marks Caclulator

ONLINE COURSE PORTAL is a comprehensive and innovative system for effective management of content, this portal practically covers all the courses and connects students with their respective course teacher.

FUNCTIONALITY

You can perform the following tasks using the OMLINE COURSE PORTAL:

1: Students

  • Can register for a course
  • Can view their academic calender and time table
  • Can view course materials
  • Can submit assignments

2: Teacher

  • Can register for a course
  • Can view their academic calender and time table
  • Can add course materials
  • Can assign assignments
  • Can update attendance
  • Can add or remove announcements

3. Admin

  • Can add/remove course
  • Can approve or disapprove teachers/students
